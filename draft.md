# NER Experiments Draft

| # | Experiment | Description | F1 Score |
|---|---|---|---|
| 1 | LFM Base (Standard Causal) | Frozen LFM2.5-350M base with a trainable 2-layer MLP classifier on top. Causal masking was fully preserved, meaning each token could only attend to previous tokens. | 0.5607 |
| 2 | RoBERTa Baseline | Fully fine-tuned roberta-base for 3 epochs on the CoNLL2003 dataset. Serves as the gold standard since RoBERTa is a fully bidirectional encoder-only transformer. | 0.9213 |
| 3 | LFM Bypass (Causal Hack) | Preprocessed input as `w1 w2 w3 <eos> w1 w2 w3` so the causal model sees full context in the duplicated half. Evaluated metrics on the second sequence where context was already processed. | 0.6915 |
| 4 | LFM Unmasked (Last 2 Layers) | Patched the model forward to supply non-causal masks to layers 14 and 15 only. Fully fine-tuned these layers while keeping the rest frozen, proving local unmasking injects bidirectional context. | 0.8476 |
| 5 | LFM Unmasked (Last 4 Layers) | Extended architectural unmasking to layers 12 through 15 with full fine-tuning. Demonstrated that deepening the bidirectional receptive field directly improves token classification quality. | 0.8739 |
| 6 | LFM Unmasked (8 Layers + LoRA) | Unmasked the entire late half of the network (layers 8-15) and applied LoRA (r=16, alpha=32) across all layers for parameter-efficient tuning. Achieved near-parity with RoBERTa while preventing catastrophic forgetting. | 0.8992 |
| 7 | LFM Unmasked (12 Layers + LoRA Optimized) | Pushed further by unmasking 12 layers (4-15) with higher LoRA capacity (r=64, alpha=128) over 4 epochs. Performance regressed slightly, suggesting unmasking too early disrupts the pre-trained feature hierarchy. | 0.8799 |
| 8 | LFM Unmasked (Middle 8 Layers + LoRA) | Unmasked layers 4-11 with LoRA (r=16, alpha=32), keeping both early and late layers causal. Underperformed the last-8 variant because causal layers after the unmasked zone squash bidirectional signal before the classifier. | 0.8836 |
| 9 | LFM Unmasked (Last 8 Layers Full FT) | Unmasked layers 8-15 with full fine-tuning, first 8 layers frozen. No LoRA used. Performed worse than LoRA, confirming that updating too many parameters causes catastrophic forgetting of pre-trained representations. | 0.8677 |
| 10 | LFM Gradient LoRA (Last 8 Unmasked) | Unmasked layers 8-15, applied gradient LoRA ranks: layers 0-3 (r=4), 4-7 (r=16), 8-11 (r=32), 12-15 (r=64). Increasing capacity toward later layers. Performed essentially same as uniform r=16, suggesting gradient ranks don't add benefit over uniform LoRA for this task. | 0.8999 |
| 11 | LFM Unmasked (Last 8 Layers + LoRA + CRF) | Added a Conditional Random Field layer on top of the emissions head to enforce valid BIO label transitions. Failed to train — produced negative loss and CUDA index errors due to label masking incompatibilities with the CRF library. | Failed |
| 12 | LFM Unmasked (8 Layers + LoRA, Base Model) | Same setup as experiment 6 but evaluated on the raw pre-trained base model without any prior SFT. Performed slightly better by 0.5 points, suggesting the pre-trained representations already contain sufficient bidirectional-compatible features for NER. | ~0.904 |
| 13 | Gemma 3 Unmasked (Last 8 Layers + LoRA) | Unmasked layers 10-17 of Gemma 3 270M, patched forward to inject custom 4D non-causal mask. Trained with LoRA (r=16, alpha=32). Reached 0.9023 F1 score, demonstrating high baseline capacity. | 0.9023 |
| 14 | Gemma 3 Unmasked (Last 8 Layers + LoRA + SwiGLU) | Replaced the standard GELU classification head with a custom SwiGLU architecture. Performance slightly regressed compared to standard GELU, suggesting the added capacity at the classification head might lead to mild overfitting or requires further tuning. | 0.8974 |
